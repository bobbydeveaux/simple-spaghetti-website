# Simple Spaghetti Website

A minimal HTML website displaying "I love spagheeti" - no CSS, no JavaScript, just pure HTML.

## Features

- Single `index.html` file with valid HTML5 structure
- Displays the message "I love spagheeti"
- Minimal file size (125 bytes)
- No external dependencies
- Works in all modern browsers

## Usage

Open `index.html` in any web browser or serve it with any web server.
