# Simple Spaghetti Website

This repository contains both a React frontend for pasta recipes and a Python FastAPI backend for authentication.

## Components

### Frontend (React)
- React application for browsing pasta recipes
- Located in `src/` directory
- Built with Vite and styled with Tailwind CSS

### Backend (Python Auth API)
- FastAPI-based authentication API with JWT token management
- Located in `api/` directory
- See `api/README.md` for detailed setup and usage instructions

## Quick Start

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
pip3 install -r requirements.txt
python3 -m api
```

For more detailed setup instructions, see the respective README files in each component directory.
